Avisos Importantes!

Este projeto visa AUXILIAR o aprendizado de linguagem assembly para os alunos do curso TADS na disciplina de Arquitetura de Computadores.

Este projeto encontra-se em fase de desenvolvimento, portanto nem todos os recursos estão implementados!

Caso o emulador apresente algum 'BUG', por favor, enviar o problema para Victor Carlquist ou Matheus Liberato.

Observações

  • O emulador ainda não suporta números em formato hexadecimal e em formato binário;

  • O Rotulo não deve ser informado na mesma linha que a instrução;

    Exemplo de código:
    LDI R16,5
    CPI R16,4
    BRGE PULA ;verifica se R16 >= 4
    LDI R18,55
    PULA:
    LDI R20,5


Lista dos Comandos Implementados:

.ORG ADD ADC ADIW ASR LDI CPI CP BRGE BREQ BRLT BRSH BRCC BRLO BRCS BRMI BRVS BRNE MOV NEG ST STS LD LDS BRPL BRVC JMP
